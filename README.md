# 🎌 Site sobre Animes

Projeto acadêmico desenvolvido para a disciplina de **Programação Web**.
O site apresenta informações sobre três animes populares:

* One Piece
* JoJo's Bizarre Adventure
* Frieren: Beyond Journey's End

O objetivo do projeto é praticar **HTML semântico e CSS**, além da estrutura básica de uma página web.

---

# 📚 Conteúdo do Site

O site possui as seguintes seções:

* **Header** – Título do site.
* **Menu de navegação** – Links internos para navegar pelas seções.
* **Seção One Piece** – Resumo e motivos para assistir.
* **Seção JoJo's Bizarre Adventure** – Curiosidades e características da obra.
* **Seção Frieren** – Destaques da narrativa e temas do anime.
* **Tabela comparativa** – Informações como ano de lançamento, gênero e tema.
* **Vídeo incorporado** – Trailer de anime.
* **Formulário de contato** – Campos de nome, email e mensagem.
* **Footer** – Informações do projeto.

---

# 🛠 Tecnologias Utilizadas

* **HTML5**
* **CSS3**

Principais conceitos aplicados:

* Estrutura semântica (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
* Links internos
* Listas ordenadas e não ordenadas
* Tabelas
* Formulários
* Incorporação de vídeo com `iframe`
